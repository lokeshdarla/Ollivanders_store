# Ollivanders Store
Ollivanders Store is a wizard-themed e-commerce website where fans of the Harry Potter series can explore and purchase a variety of magical wizard wands.

```
/src
|-- /assets

|   |-- /product

|       |-- ProductDetail.tsx
|   |-- Cart.tsx
|   |-- Checkout.tsx
|-- /pages

|-- index.tsx
|-- /public
    |-- index.html

```
